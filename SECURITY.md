# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest release | Yes |
| Older releases | No |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly:

1. **Do not** open a public issue
2. Email **security@mcnerve.com** or use [GitHub's private vulnerability reporting](https://github.com/McNerve/runewise/security/advisories/new)
3. Include steps to reproduce and the potential impact
4. You'll receive a response within 48 hours

## Scope

RuneWise is a desktop companion app that:
- Makes read-only API calls to public OSRS services (Hiscores, Wiki, Wise Old Man)
- Stores no credentials or authentication tokens
- Runs entirely locally with no server component
- Does not interact with the OSRS game client
